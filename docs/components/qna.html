Apakah Upam Pro cocok untuk blog kecil maupun website organisasi?

Ya. Upam Pro dirancang fleksibel: bisa dipakai untuk blog personal, media berita kecil, hingga website organisasi yang membutuhkan pemisahan antara artikel, halaman statis, dan pengelolaan media.

Apa perbedaan antara modul Write dan modul Manage?

Modul Write fokus pada proses penulisan dan pengeditan satu artikel. Modul Manage fokus pada daftar semua konten yang sudah dibuat, sehingga Anda bisa melihat gambaran besar, mengubah status, atau melakukan pengelolaan massal lebih cepat.

Bagaimana cara menambahkan gambar ke artikel?

Biasanya alurnya: upload gambar terlebih dahulu di modul Gallery, lalu gunakan URL atau mekanisme sisip media yang tersedia di modul Write. Anda dapat menyesuaikan cara penempatan gambar sesuai tema atau kebutuhan layout Anda.

Apakah saya bisa mengatur siapa saja yang boleh mengakses panel admin?

Bisa. Modul Profiles digunakan untuk membuat dan mengelola akun pengguna, sekaligus membatasi peran dan hak akses. Dengan cara ini, hanya akun tertentu yang diizinkan untuk mengubah konten, pengaturan, atau bagian kritis lain.

Apakah Upam Pro bisa diintegrasikan dengan sistem lain?

Modul Settings menyiapkan ruang untuk konfigurasi API dan koneksi ke layanan eksternal. Detail integrasi dapat Anda sesuaikan sendiri di sisi implementasi sesuai kebutuhan proyek.

Bagaimana jika saya ingin mengubah tampilan tanpa mengganggu data?

Gunakan modul Wizard untuk memasang atau mengganti template tampilan. Tujuannya agar perubahan UI dapat dilakukan tanpa mengubah struktur data konten yang sudah ada.
